🚖 Ola Data Analysis (SQL Project)

📊 Project Overview

This project analyzes Ola ride booking data using SQL queries.

📁 Dataset

The dataset contains information about:

* Bookings
* Customers
* Drivers
* Ride status
* Payment methods

🔍 Key Analysis Performed

 1. Successful Bookings

Retrieve all successful rides.

 2. Average Ride Distance

Find average ride distance per vehicle type.

 3. Cancelled Rides

Count rides cancelled by customers and drivers.

 4. Top Customers

Identify top 5 customers with highest bookings.

5. Driver Issues

Analyze cancellations due to driver-related issues.

 6. Ratings Analysis

Find max and min driver ratings.

 7. Payment Analysis

Filter rides paid using UPI.

 8. Customer Ratings

Average customer rating per vehicle.

9. Revenue Analysis

Total booking value of successful rides.

 10. Incomplete Rides

List incomplete rides with reasons.

 🛠 Tools Used

* SQL (Structured Query Language)
